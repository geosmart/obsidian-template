# MECE 原则

---

## Who（适用对象）
- 管理咨询顾问
- 数据分析师
- 流程优化专家

## When/Where（应用场景）
- 市场细分研究
- 组织结构设计
- 问题树构建

## How（使用步骤）
1. 确定分类维度
2. 检查独立性（无重叠）
3. 验证穷尽性（全覆盖）
4. 迭代优化分类
