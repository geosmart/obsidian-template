# MECE 原则 [[思维模型]]

## What（定义）
相互独立 (Mutually Exclusive) 完全穷尽 (Collectively Exhaustive) 的分类原则

## Why（价值）
- 确保分类的严谨性
- 避免重复和遗漏
- 适用于结构化分析

## Who（适用对象）
- 管理咨询顾问
- 数据分析师
- 流程优化专家

## When/Where（应用场景）
- 市场细分研究
- 组织结构设计
- 问题树构建

## How（使用步骤）
1. 确定分类维度
2. 检查独立性（无重叠）
3. 验证穷尽性（全覆盖）
4. 迭代优化分类
