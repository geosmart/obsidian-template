# 六顶思考帽 [[思维模型]]

## What（定义）
通过六种颜色帽子代表不同思考维度的平行思维工具

## Why（价值）
- 促进全面思考
- 减少会议冲突
- 适用于团队决策

## Who（适用对象）
- 团队领导者
- 会议主持人
- 项目协调人

## When/Where（应用场景）
- 重大决策会议
- 跨部门协作
- 创意方案讨论

## How（使用步骤）
1. 白帽：陈述客观事实
2. 红帽：表达情感直觉
3. 黑帽：风险预警分析
4. 黄帽：积极价值发现
5. 绿帽：创意提案发散
6. 蓝帽：流程控制总结
